# 01-principles — 原理原則

ルール・ステージ・ブキ非依存の普遍理論

## ページ一覧

- [打開の基本方針](dakai-fundamentals.md) — 打開を行う際は、抑え側のチームが有利なポジションで待ち構えていることを認識しなければなりません。
- [復帰状況の把握](enemy-respawn-prediction.md) — 戦況を正確に捉えるためには、まず「誰がいつ倒されたか」を把握する必要があります。
- [立ち回りモードの明確な切り分け](engagement-mode-switching.md) — 状況に応じて、「塗り・ヘイト稼ぎ」と「潜伏・キル」の2つのモードを明確に使い分ける必要があります。
- [遮蔽物との距離感と視界の確保](peeking-and-clearing.md) — 遮蔽物から体を出す際、壁に密着した状態で最短距離を移動するのは必ずしも最善ではありません。
