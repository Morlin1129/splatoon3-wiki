# 01-principles — 原理原則

ルール・ステージ・ブキ非依存の普遍理論

## ページ一覧

- [自陣（手前）処理の重要性](2026-04-26-general-dakai-home-base-clearing.md) — 打開の基本は、自陣付近（手前）に潜伏している敵の処理から確実に始めることである。
- [役割の重複を避ける判断](2026-04-26-general-osae-waiting-actions.md) — 味方が正面ルートの敵を十分に抑えられていると感じた場合、同じ場所に留まって射撃を続けるのは得策ではありません。
- [前線確保の優先順位](2026-04-26-general-pushing-frontline-priority.md) — 敵が自陣に侵入していないことが確実であり、チームとして攻め込むべき場面では、自陣付近を丁寧に塗る「手前の塗り」よりも、**素早く前線を上げて敵に圧力をかけること**を最優先します。
- [打開の基本方針](dakai-fundamentals.md) — 打開を行う際は、抑え側のチームが有利なポジションで待ち構えていることを認識しなければなりません。
- [復帰状況の把握](enemy-respawn-prediction.md) — 戦況を正確に捉えるためには、まず「誰がいつ倒されたか」を把握する必要があります。
- [立ち回りモードの明確な切り分け](engagement-mode-switching.md) — 状況に応じて、「塗り・ヘイト稼ぎ」と「潜伏・キル」の2つのモードを明確に使い分ける必要があります。
- [遮蔽物との距離感と視界の確保](peeking-and-clearing.md) — 遮蔽物から体を出す際、壁に密着した状態で最短距離を移動するのは必ずしも最善ではありません。
